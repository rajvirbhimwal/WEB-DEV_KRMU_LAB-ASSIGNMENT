# JavaScript Console Quiz

A simple quiz game that runs in your web browser.

## Files
* `index.html` - The main page to start the game.
* `script.js` - The code containing questions and game logic.

## How to Play
1. Download both files to the same folder.
2. Open `index.html` in your browser (Chrome, Firefox, etc.).
3. Click the **"Start / Restart Quiz"** button.
4. Type your answers in the pop-up box.

## Features
* **7 Questions:** Increasing difficulty from Easy to Legendary.
* **Score Tracking:** Shows your final score at the end.
* **Replay:** You can play again without refreshing the page.
